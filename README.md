# kb-app

kb-appはEC機能付きのアートポートフォリオです。<br>
自分が通う床屋の担当が、アート作品などをwebで発信したい思いを抱いていた為、<br>
作成したアプリです。
彼が手掛けるアートの一覧、商品出品、購入機能までが実現されています。
